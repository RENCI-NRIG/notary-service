Frequently Asked Questions

What does Role mean?


The user's Role determines their ability to create, read, update and delete various items within Notary Service. Role's are selected from the Preferences section of the user's profile page. A user can only operate as one role at a time. If a user has multiple roles available to them, they can switch between the available roles to accomplish different tasks.

Possible Roles Privileges
Administrator Full administrative rights over notary service deployment
- Can create, read, update, and delete templates
- Can create, read, update, and delete datasets
- Can create, read, update, and delete infrastructure
- Can create, read, update, and delete projects
- Can create, read, update, and delete workflows
Data Provider Can create, read, update, and delete templates
Can create, read, update, and delete datasets
Per project association where dataset is used
- Can read templates
- Can read datasets
- Can read infrastructure
- Can read projects
- Can complete workflows
Infrastructure Provider Can create, read, update, and delete infrastructure
Per project association where infrastructure is used
- Can read datasets
- Can read infrastructure
- Can read projects
- Can complete workflows
Institutional Governance Per project association where institution/organization contains PI member
- Can create read datasets
- Can create read infrastructure
- Can create read projects
- Can create complete workflows
Project Member Per project association that contains Project Member
- Can read datasets
- Can read infrastructure
- Can read projects
- Can complete workflows
Project PI Per project association that contains Project PI
- Can read datasets
- Can read infrastructure
- Can read projects
- Can complete workflows
Project PI Admin Can create, read, update, and delete projects
Can create, read, update, and delete workflows
Per project association that contains Project PI Admin
- Can read templates
- Can read datasets
- Can read infrastructure
- Can read projects
- Can complete workflows
No Role Access limited to About, FAQ and profile page
